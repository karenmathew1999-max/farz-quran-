# Contributing to Farz Quran

Thank you for your interest in contributing to the Farz Quran project! This document provides guidelines and instructions for contributing.

## 🎯 Code of Conduct

We are committed to providing a welcoming and inspiring community for all. Please read and adhere to our code of conduct:

- Be respectful and inclusive
- Welcome newcomers and help them get started
- Focus on what is best for the community
- Show empathy towards other community members

## 🚀 How to Contribute

### Reporting Bugs

If you encounter a bug, please:

1. Check if the issue already exists
2. Create a new issue with:
   - Clear, descriptive title
   - Detailed description of the problem
   - Steps to reproduce
   - Expected vs. actual behavior
   - Your environment (OS, browser, etc.)

### Suggesting Enhancements

We love feature suggestions! Please:

1. Check if the feature has already been suggested
2. Provide a clear, detailed description
3. Explain the use case and benefits
4. Include examples if possible

### Submitting Pull Requests

1. **Fork the repository**
   ```bash
   git clone https://github.com/karenmathew1999-max/farz-quran-.git
   cd farz-quran-
   ```

2. **Create a feature branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Make your changes**
   - Follow the existing code style
   - Write clear, concise commit messages
   - Keep commits focused and logical

4. **Test your changes**
   - Run all tests locally
   - Ensure no breaking changes

5. **Push and create a Pull Request**
   ```bash
   git push origin feature/your-feature-name
   ```

## 📋 Pull Request Guidelines

- Provide a clear description of what you're changing and why
- Reference any related issues
- Include screenshots for UI changes
- Ensure all CI checks pass
- Request review from maintainers

## 🎨 Coding Standards

- Follow existing code conventions
- Write clean, maintainable code
- Add comments for complex logic
- Use meaningful variable and function names
- Keep functions focused and modular

## 📚 Documentation

- Update relevant documentation
- Add comments to complex sections
- Include examples for new features
- Keep README up-to-date

## ❓ Questions?

Feel free to:
- Open an issue for discussions
- Reach out to maintainers
- Join our community discussions

## 🙏 Thank You!

Your contributions make Farz Quran better for everyone. We appreciate your effort and support!
